A Platform Game Prototype: Shadow Walker

Character has special skills to solve the puzzle and achieve to the end.

Game demo available at: Github

Skill Description

Skill One

Skill Two

player turn into shadow mode (blue icon). This will allow you to go through all items.

when player in red skill mode, player can rotate any rotatable item which is surrounding the player.

How to use

How to use

hold left mouse button to turn into shadow mode, when player release left mouse button, stop using skill one.

You can find the items (that are currently rotatable) by moving the mouse to an item, if that item is rotatable, there will be a red point show up. When there is red point, Player can rotate that item by holding left mouse button. When player release the button, the rotation will finish.

Restriction

Restriction

player can't collect points in shadow mode.

player cannot rotate the item that player is currently standing on.
